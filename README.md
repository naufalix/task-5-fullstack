
## Instalasi
- Clone Repository
- ```composer update```
- ```composer install```
- ```npm install```
- Buat database dengan nama ```okegarden```
- ```php artisan key:generate```
- ```php artisan migrate```
- ```php artisan db:seed```
- ```php artisan serve```
- ```npm run dev```
